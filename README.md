# Job Role Prediction using Machine Learning

## Overview

This project predicts the most suitable job role based on a user's skills using a Machine Learning model. The application is built with Python and Streamlit, allowing users to enter their skill ratings and receive a predicted job role.

## Features

* Predicts suitable job roles based on skill inputs.
* User-friendly Streamlit interface.
* Random Forest Classifier for prediction.
* Fast and accurate predictions.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit
* Matplotlib
* Joblib

## Files

* `job_app.py` – Streamlit application.
* `job_role_prediction.py` – Model training script.
* `job_role_model.pkl` – Trained machine learning model.
* `label_encoder.pkl` – Label encoder for decoding predicted job roles.
* `dataset9000.csv` – Dataset used for training.
* `requirements.txt` – Required Python libraries.

## Installation

1. Clone the repository:

```bash
git clone <repository-link>
```

2. Install the required libraries:

```bash
pip install -r requirements.txt
```

3. Run the Streamlit application:

```bash
streamlit run job_app.py
```
